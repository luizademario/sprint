Sobre o projeto

Nascido da necessidade real dos estudantes

O JOVI surgiu a partir de um mapeamento das dores dos estudantes: fotos espalhadas, áudios perdidos, matérias misturadas. Nossa proposta é simples — deixar a tecnologia trabalhar para você, enquanto você foca no que importa: aprender.

  • ✦ Interface limpa e focada
  • ✦ IA embarcada no dispositivo
  • ✦ Baixo consumo de bateria
  • ✦ Funciona sem internet
