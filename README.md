> Sprint 2 – Web Development & Front-End

---

## 📱 Sobre o Projeto

O **JOVI** é uma câmera inteligente com suporte de inteligência artificial, desenvolvida para estudantes. O aplicativo reconhece, organiza e integra conteúdos multimídia (imagem, áudio e texto), permitindo busca avançada e facilitando a revisão das informações.

### Funcionalidades
- 🎯 **Modo Estudante** – organização automática por matéria via IA
- 🎙️ **Transcrição de Áudio** – transcrição automática em tempo real
- 🔍 **Busca Inteligente** – OCR em imagens e busca por palavra-chave
- 🌐 **Tradução Automática** – tradução de texto presente nas fotos
- 🔗 **Integração Foto + Áudio** – vinculação automática de mídia simultânea

## ✅ Requisitos Atendidos

- [x] Manipulação dinâmica de elementos e eventos das páginas
- [x] Validação de formulários e login (login, cadastro, contato)

- [x] Layout dinâmico e moderno (dark theme, animações, micro-interações)
- [x] HTML semântico (`<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`, `<aside>`, `<header>`)
- [x] Flexbox para posicionamento e estrutura dos conteúdos
